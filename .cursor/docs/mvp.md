# MVP Scope

The MVP focuses on building trust between farmers and consumers.

The goal is NOT to build a full ecommerce platform.

# MVP Features

## Authentication
- register
- login

## Farmer Profiles
- profile image
- bio
- location
- farming philosophy

## Product Listings
- title
- description
- photos
- simple pricing
- seasonal availability

## Product Storytelling
- videos associated with products
- farming updates
- growth timeline

## Feed
- discover farmers
- discover products
- recent updates

## Reviews
- farmer reviews
- product reviews

# NOT Included In MVP

Do NOT build:
- payments
- delivery system
- subscriptions
- advanced moderation
- AI features
- livestreaming
- complex verification
- advanced analytics
- admin dashboard
- notifications
- podcasts
- messaging system

# MVP Goal

The MVP should answer one question:

"Do people enjoy discovering and trusting local farmers through authentic content?"
